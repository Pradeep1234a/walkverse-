# Liquid Glass Calculator (Squircle Edition)

A premium flagship Android calculator inspired by the classic iPhone layout and Apple's futuristic **iOS 26 Liquid Glass** design language.

## 💎 Features
- **Highly Transparent Optical Glass**: Zero milky diffusion or heavy frosted overlays; the background remains fully visible.
- **Continuous 8s Specular Reflections**: Animated light streaks and glares glide across button faces on a loop.
- **Parametric Superellipses**: All key shapes use a mathematically precise superellipse ($n = 4.8$) rather than standard rounded rectangles.
- **Edge Highlights & Glare**: Double-rim borders (rim highlights scaling to 80% opacity at the top edge).
- **Result Prefixing**: Calculated results dynamically display with a leading `=` (e.g. `=12,454`).
- **Dynamic Commas**: Display inputs format automatically with thousands separators.
- **Swipe-to-Delete**: Swipe left or right on the main display panel to erase digits.
